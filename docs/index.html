Why Mangadeck

Everything you need in one tool

No more switching between sites. No more missing chapters. No more broken downloads.

merge_type

Multi-Source Aggregation

If MangaDex stops at chapter 763, Mangadeck automatically finds chapters 764-1076 from Manganato, MangaKakalot, and others. You get a complete, gap-free download every time.

devices

Three Interfaces

A full web dashboard with live progress, a desktop GUI built with Tkinter, and a rich terminal UI with animated progress bars. Use whatever fits your workflow.

file_present

Multiple Formats

Export as EPUB (Apple Books compatible), CBZ (comic readers), PDF, or organized image folders. Proper metadata, reading direction, and cover pages included.

crop

Smart Auto-Crop

Automatically detects and removes white or black borders from pages. Configurable threshold and minimum ratio prevents over-cropping. Your pages look clean on every device.

swap_horiz

RTL / LTR / Vertical

Auto-detects reading direction for manga (right-to-left), manhwa (vertical), and manhua (vertical). Embeds proper metadata so your reader app does the right thing.

notifications_active

Notifications

Get notified when downloads finish via desktop push notifications, Discord webhooks, or Telegram bots. Perfect for large batch downloads overnight.

shopping_cart

Cart & Queue

Browse and add manga to a cart, then download everything at once with a single click. Priority queue processes items sequentially with configurable delays.

api

REST API

Full headless server mode with REST endpoints and Server-Sent Events for live progress. Build your own integrations, scripts, or frontends on top of Mangadeck.

Get Started

Up and running in 30 seconds

bash 
# Clone the repo
$ git clone https://github.com/Compromisee/Mangadeck.git
$ cd Mangadeck

# Install dependencies
$ pip install -r requirements.txt

# Launch the web dashboard
$ python main.py
  Mangadeck Dashboard: http://127.0.0.1:5000

# Or use the terminal UI
$ python main.py --mode tui

# Or the desktop GUI
$ python main.py --mode gui
